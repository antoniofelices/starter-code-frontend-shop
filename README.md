# Sprint 2.2 IT Academy | Shop

## Description

A basic e-commerce.

## Technologies

-   JavaScript.
-   Bootstrap.
-   Vite.

## Requirements

-   Node.js v23.9.0
-   npm v11.2.0

## Documentation

### Installation

1. Clone this repo

```bash
$ git clone git@github.com:antoniofelices/starter-code-frontend-shop.git
```

2. Install packages

```bash
$ npm run install
```

3. Run develop

```bash
npm run dev
```
